======================================
   코딩 멘토 서버 사용 방법
======================================

📋 한 번 설정하면 끝! 더 이상 매번 설정할 필요 없음

1️⃣ 자동 시작 설정 (한 번만 실행)
   ▶️ install-autostart.bat 실행 (관리자 권한으로)
   → 컴퓨터 켤 때마다 자동으로 서버 시작

2️⃣ 수동 시작 (필요시)
   ▶️ start-server-auto.bat 실행
   → IP 주소가 바뀌어도 자동으로 감지하고 설정

3️⃣ 접속 방법
   집에서: 화면에 표시되는 주소 사용
   학원에서: 같은 주소 사용 (IP가 동일하게 유지됨)

======================================
🔧 문제 해결

Q: 학원에서 접속이 안 돼요
A: 공유기 설정에서 포트 포워딩 필요
   - 192.168.68.1 접속 (공유기 관리)
   - 포트 포워딩: 3001, 3006 포트 열기

Q: IP가 자꾸 바뀌어요  
A: start-server-auto.bat 사용하면 자동으로 감지

Q: 서버가 자동으로 안 켜져요
A: install-autostart.bat를 관리자 권한으로 실행

## 💰 **DCA 공식 암기카드**

### 📈 **목표 수익률 → 매수 기간**
```
🎯  50% = 0.5달 = 2주
🎯 100% = 1달 = 4주  
🎯 200% = 2달 = 8주
🎯 300% = 3달 = 12주
🎯 500% = 5달 = 20주
🎯 600% = 6달 = 24주
```

### 💡 **공식**
```
📊 목표수익률(%) ÷ 100 = 매수기간(달)
📅 매수기간(달) × 4 = 매수기간(주)
```
